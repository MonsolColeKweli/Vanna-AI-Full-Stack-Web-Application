About The Project

VANNA itself represents the culmination of advancements in natural language processing, database interfacing, and AI-driven automation. It brings together the power of large language models, schema-aware query generation, and RESTful integration to make complex data retrieval as simple as asking a question—bridging the gap between human intent and machine-readable commands.

Project Introduction: Vanna Smart Database Frontend Application

For our final project in CSE383, we developed a responsive, full-stack web application that queries natural language with structured database responses using modern AI and web technologies such as Vanna. The centerpiece of our project is an intelligent frontend that interacts with the Vanna API—a tool capable of transforming plain English questions into precise SQL queries. This allows users to extract meaningful data from a MySQL database without needing to write a single line of code. The application is built using Docker to containerize and manage different components—like our Apache web server, PHP REST API, and the Vanna backend—all hosted on our personal OpenStack instance. Our website includes five main sections: a landing page, two team member pages, a dynamic AI-powered query interface, and a searchable transaction history log. Through the use of HTML, CSS, JavaScript, Bootstrap, jQuery, and AJAX, we ensured the site is not only visually engaging but also responsive across different devices. Users interact with the application by entering natural language questions through our custom frontend. These queries are processed in real time by the Vanna backend, which translates them into SQL, fetches the corresponding data, and returns structured answers. Simultaneously, all user interactions are logged through a student-built PHP REST API into a SQLite database, allowing for complete historical traceability via an interactive history page. This project showcases our ability to combine cloud infrastructure, modern web development techniques, and artificial intelligence into a cohesive and user-friendly product—culminating in a system that makes data querying accessible to both technical and non-technical users alike.
